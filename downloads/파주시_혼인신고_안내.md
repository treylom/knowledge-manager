# 파주시 혼인신고 안내

> 출처: 파주시 민원 콜센터 통화 (2026.03.09)

---

## 준비물 체크리스트

- [ ] **신분증** 2매 (두 분 각각) - 임시 신분증도 가능
- [ ] **혼인신고서** (아래 링크에서 다운로드 또는 시청 현장 비치)
- [ ] **증인 2명**의 인적사항 및 서명 (혼인신고서 증인란에 미리 작성)

## 증인 요건

- 성인이면 누구나 가능
- 가족, 친구, 지인 모두 가능
- 2명 필요
- 증인의 인적사항과 서명을 **정자**로 기입

## 방문 안내

| 항목 | 내용 |
|------|------|
| **장소** | 파주시청 민원실 |
| **운영시간** | 평일 근무시간 (점심시간 교대근무, 대기 있을 수 있음) |
| **야간 민원** | 매주 목요일 18:30 ~ 21:00 (야간 민원실) |
| **단독 신고** | 가능 (단, 두 분의 신분증 모두 지참 필수) |
| **전화번호** | 031-940-4114 |
